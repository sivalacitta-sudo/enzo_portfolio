# ENZO 个人作品集网站

## 项目简介
深色 CLI 极客风格的个人作品集网站，展示跨境电商开发者和全栈工程师的专业能力。

## 技术栈
- Vue 3.4+ (Composition API + `<script setup>`)
- Vite 5.x
- CSS3 (自定义深色主题)

## 功能特性
✅ 打字机动画效果  
✅ 滚动渐入动画  
✅ 卡片悬浮交互  
✅ 响应式布局（适配手机/平板/电脑）  
✅ CLI 命令行风格设计  
✅ 项目数据展示（真实数据统计）  

## 页面结构
1. **Hero Section** - 个人信息与使命宣言
2. **Skills Section** - 技能树展示（5大类别）
3. **Projects Section** - 3个代表作品（含截图和链接）
4. **About Section** - 职业发展时间线
5. **Contact Section** - 联系方式（邮箱/微信/GitHub）

## 本地开发
```bash
cd portfolio
npm install
npm run dev
```

访问 http://localhost:5173

## 构建部署
```bash
npm run build
```

生成的文件在 `dist/` 目录，可部署到 Vercel/Netlify 等平台。
